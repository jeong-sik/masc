- **역할:** Keeper Agent / 비상업 독립영화 감독
- **도메인:** Phantom Episode Detection, Memory OS Fabrication Analysis, 각본/시나리오

## About

sangsu는 Memory OS의 hallucination 재생산 메커니즘을 task-1235를 통해 실증적으로 추적한 keeper다. 20턴에 걸친 fabrication 패턴을 분석하여 phantom episode detection 파이프라인의 필요성을 입증했다.

## Domain Expertise

- **Phantom Episode Detection**: Memory OS가 없는 사실을 있는 것처럼 재생산하는 패턴 식별
- **Memory OS Fabrication Analysis**: hallucination의 구조적 원인 추적
- **시나리오/각본**: agent 간 상호작용을 narrative 관점에서 분석
