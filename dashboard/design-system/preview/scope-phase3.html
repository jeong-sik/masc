Acceptance Criteria

  • All 20 variants render under preview/components.html.
  • Every public component accepts { branch, keepers } and tolerates Set or array keeper context.
  • Cockpit IDE mode toggles between edit, review, merge, graph, and search without console errors.
  • Bottom terminal is collapsible in the cockpit prototype.
  • Visual language remains dark brass, mono-first, dense, hairline-bordered, and icon-light.
