# Tool Readiness Notification Channel

## 1. Retirement

This draft depended on a credential readiness model that no longer exists.
MASC does not emit repository GitHub credential readiness transitions because it
does not own repository GitHub identity materialization.

## 2. Replacement Contract

Sandbox readiness may still be surfaced by ordinary keeper/tool telemetry, but
there is no credential transition channel for repository GitHub access.

This RFC remains only as the retired record of the deleted proposal.
