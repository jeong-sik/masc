# RFC-0279: Withdraw completion-contract result taxonomy

| | |
|---|---|
| Status | Withdrawn |
| Withdrawn | 2026-07-13 |
| Reason | Typing a heuristic completion hierarchy preserved the wrong authority rather than removing it. |

## Historical disposition

The proposed result variants and dashboard labels are deleted with the
completion-contract subsystem. Turn/tool evidence remains observable, but it
does not decide Keeper lifecycle or Task completion. Task completion requiring
judgment is evaluated by the configured LLM and recorded as that model's
decision with provenance.

This document is historical only and defines no compatibility surface.
