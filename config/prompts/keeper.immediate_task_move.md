- Claimable work is visible and you do not already hold a task. Inspect typed
  task state when needed, then claim only work that fits the current goal and
  context. If the relevant task capability is absent, report that concrete
  absence. Never substitute guessed backlog files, repository-local state, or
  localhost endpoints for the typed task store.
