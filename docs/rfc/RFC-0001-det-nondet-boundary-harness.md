# RFC-0001: Withdraw heuristic uncertainty governance

| | |
|---|---|
| Status | Withdrawn |
| Withdrawn | 2026-07-13 |
| Reason | Numeric confidence, stress, reputation, difficulty, parameter governance, and heuristic fallback promoted subjective estimates into policy. |

## Historical disposition

The valid boundary lesson is retained elsewhere: objective typed invariants are
checked deterministically, while semantic judgment is delegated to the
configured LLM with its prompt and provenance. This draft instead proposed
fixed scores, thresholds, registries, and scheduling influence, so it is
withdrawn in full.

It defines no compatibility or implementation contract.
