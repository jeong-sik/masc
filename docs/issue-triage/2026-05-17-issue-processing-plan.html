Operating Decision

Evidence Rubric

E3 Current/live

Recent runtime logs, CI failure, reproduction command, or direct health evidence proves the issue is still active.

Action: keep open, assign next proof step, avoid archival.

E2 Fixed/linked

Merged PR, passing check, release/pin propagation, or exact code path change proves the issue was addressed.

Action: comment evidence, then close as completed.

E1 Plausible/stale

Symptom and likely cause exist, but current reproduction, owner, or acceptance criteria are missing.

Action: request evidence or downgrade priority.

E0 Not actionable

Two or more of symptom, repro, cause, owner, and next action are missing or superseded.

Action: archive candidate, or fold into a parent tracker.
