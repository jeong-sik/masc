This path is retained only as a decision tombstone. The draft described the
removed tension/meta-cognition subsystem and never became a current Keeper
contract. No runtime, prompt, persistence, or dashboard consumer may depend on
it.

[`KEEPER-STATE-OWNERSHIP.md`](../KEEPER-STATE-OWNERSHIP.md). The other RFC using
number 0027, [`RFC-0027-capability-typed-runtime.md`](./RFC-0027-capability-typed-runtime.md),
