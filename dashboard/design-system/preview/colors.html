Usage Rules

The short list
Do

Use brass for one and only one element per view: the active row, the focused tab, the primary button. Use status colors for data. Use fg-3 for all labels.

Don't

Never use brass for hover. Never use pure white for text. Never saturate status colors past their base — no neon red, no electric green. No gradients on data surfaces.
