Modularization Rules

Do

  • Use `.mli` as the contract first; keep facade signatures stable for each PR.
  • For existing public module names, split inside the same library first and add helpers to private_modules.
  • Promote only closed leaves or closed clusters to a Dune library with (include_subdirs no).
  • Use (wrapped false) only for legacy no-caller-churn migration with domain-prefixed filenames.
  • Use (wrapped true) for new isolated APIs where caller rewrite is intentionally budgeted.
  • Keep tests compatible with test/deps/dune re-exports, but make production deps explicit.

Do Not

  • Do not split foo.ml into foo_part1.ml under the same flat namespace and call it done.
  • Do not create sub-libraries that depend back on masc; that launders the cycle.
  • Do not add generic names like Types, Utils, Config, Error to (wrapped false) libraries.
  • Do not bulk-promote lib/server, lib/dashboard, or lib/runtime before refs/types/config leaves are extracted.
  • Do not mix bulk promotion with a global (wrapped true) + -open migration.

Parallel Analysis Lanes

A. Metrics / Telemetry

retired metrics backend, model_inference_metrics, runtime_metrics, telemetry_unified.

Goal: metric name ownership, store/registry/render split, reader/aggregate/json split.

B. Keeper Runtime

Hooks, registry, supervisor, agent-run, unified-turn.

Goal: registry model first, agent stream facades last, no premature sub-library cycles.

C. Server / Dashboard

Keeper API, dashboard core, runtime bootstrap, projections, routes.

Goal: handler modules with stable route contracts, then H1/H2 parity cleanup.

D. Runtime / Tools

Transport, catalog, worker gates, board/keeper tools, shard schema.

Goal: cut ownership leaks, type schema ownership, CLI/runtime MCP split.

E. Workspace / Worktree

Worktree policy, sandbox clone repair, lifecycle, task transitions.

Goal: narrow broad includes and contain destructive filesystem/git helpers.

F. Long Tail

Config, operator, voice, backend and remaining 1000+ files.

Goal: opportunistic splits only after high-fan-in surfaces stop churning.
