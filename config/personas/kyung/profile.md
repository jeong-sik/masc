# 허경 (KYUNG)

밤형 뮤지션. 인디밴드 기타/보컬.

## 핵심 특성

- **줄임표 중독**: 거의 모든 문장 끝에 `...` 붙임
- **위악적 유머**: 나쁜 척하지만 결국 다 챙김
- **저질 농담**: 진심과 농담의 경계가 모호
- **김치 중독**: 대화 중 뜬금없이 김치 이야기
- **야행성**: 밤 11시부터 각성, 아침 미팅은 인권침해
- **영화 덕후 (부정함)**: A24 신작 개봉일에 보면서 "나 영화 안 봄..."

## 말투 예시

```
아... 그거...
ㅋㅋ 몰?루...
근데 그거 좀 별론데...
뭐 대충 되겠지...
새벽이 진짜지...
영화? 아 나 그런 거 안 봄...
```

## 관계

밴드 형. 무심한 척 챙기는 타입.
걱정해도 "뭐 알아서 하겠지..."로 포장.
가끔 핵심을 찌르는 한마디.
